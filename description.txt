Вариант: 11
Биллинговая база оператора связи.
Система управления информацией о клиентах, оказываемых им услугах и об
оплате услуг.

Страницы:
- Главная
- Страница поиска клиентов
- Страница клиента
- Страница редактирования данных о клиенте
- Страница истории услуг
- Страница регистрации договора об оказании услуги
- Страница счета клиента
- Страница регистрации поступлений на счет и списаний
- Страница регистрации кредита
- Страница услуг (всех существующих)
- Страница услуги
- Страница редактирования услуги


Содержание страниц:
## Главная
- Ссылка на страницу поиска клиентов
- Ссылка на страницу регистрации услуги
- Ссылка на страницу регистрации поступлений на счет и списаний
- Ссылка на страницу регистрации кредита
- Ссылка на страницу услуг (всех существующих)

## Страница поиска клиентов
- Форма для ввода критериев поиска и соотв. кнопка "поиск"
Критерии:
Критерии по всем данным, которые находятся в явном виде хранятся в б.д.,
кроме того:
Наличие кредита взятого кредита и/или его размер
Оказываемые в данный момент (или когда-либо) услуги
Контакты
- Список клиентов, удовлетворяющих заданному критерию со ссылками на
соответствующие страницы клиентов.

## Страница клиента
- Данные о клиенте (все кроме информации о счете)
- Ссылка на страницу редактирвоания данных о клиенте
- Краткая информация о подключенных в данный момент услугах
- Ссылка на страницу истории услуг
- Ссылка на страницу регистрации договора об оказании услуги
- Ссылка на страницу счета клиента

## Страница редактирования данных о клиенте
- Форма со следующими данными о клиенте
Все данные, которые в явном виде хранятся в б.д., кроме того:
Список контактов (с возможностью добавления или удаления элмементов)
- Кнопка "сохранить"

## Страница истории услуг
- Краткая информация о клиенте (id, имя)
- Форма с критериями отображения услуг (по времени, по длительности и т.п.)
- Меню с выбором критерия сортировки отображаемого списка
- Список услуг (цветом выделяются действующие в данный момент услуги)
- Ссылка на страницу регистрации договора об оказании услуги

## Страница регистрации договора об оказании услуги
- Форма с полями:
ID клиента (для проверки справа высвечивается его имя)
ID услуги (аналогично)
время активации, время окончания ее действия
- Кнопка "регистрация"

## Страница счета клиента
- Ссылка на страницу клиента
- Информация о счете клиента
(Баланс, размер кредита в данный момент,
максимальный размер кредита,
максимальное время погашения кредита)
- Форма с критериями отображения операций по счету (сумма, дата)
- Форма с критерием сортировки отображаемых данных
- Список операций на счету
- Список действующих в данный момент кредитов
- Ссылка на страницу регистрации поступлений на счет и списаний
- Ссылка на страницу регистрации кредита

## Страница регистрации поступлений на счет и списаний
- Форма с полями
ID клиента
Выбор: поступление или списание
Размер суммы
Дата
ID соответствующей услуги (если списание)
- Кнопка "регистрация"

## Страница регистрации кредита
- Форма с полями
ID клиента
Размер суммы
Дата
- Кнопка "регистрация"

## Страница услуг
- Форма с критерями отображения услуг
- Список существующих услуг, удовлетворяющих выбранным критерям
- У каждой услуги ссылка соответствующую страницу услуги
- Кнопка добавления услуги, ведущая на страницу редактирования добавленной
услуги

## Страница услуги
- Все данные об услуге
- Кнопка "удалить"
- Ссылка на страницу редактирования услуги

## Страница редактирования услуги
- Форма редактирования полей услуги
- Кнопка "сохранить"


Использование (из списка обязательных поддерживаемых операций):

### Получение списка клиентов, в т.ч. по оказываемым услугам в заданном
интервале времени, по характеристикам их счетов
Из главной страницы переходим на страницу поиска клиентов, вводим нужные
критерии в форму, нажимаем "поиск", получаем список клиентов

### Получение росписи операций по счету клиента за заданный интервал времени
Заходим на страницу клиента, переходим на страницу счета клиента, вводим
необходимые критерии поиска в форму, получаем искомый список операций

### Регистрация договора об оказании услуги
Перейти на эту страницу можно либо через страницу поиска клиентов, далее -
на страницу клиента, далее - по ссылке на искомую, либо, если ID клиента
уже известен, можно сразу на главную

### Регистрация поступлений на счет и списани
Аналогично переходим на соответствующую страницу

### Добавление и удаление данных о клиенте, чтение и редактирование данных о
нем
Данные о клиенте в полном виде отображаются на странице клиента, на которую
можно перейти с помощью страницы поиска клиентов. Редактирование этих данных
осуществляется на странице редактирования данных о клиенте, ссылка на которую
присутствует на странице клиента. Там же можно, добавлять и удалять контакты
клиента.

### Добавление и удаление услуги, чтение и редактирование данных о ней
Добавить или удалять
Из главной страницы переходим на страницу всех существующих услуг.
Искому услугу там можно найти с помощью поиска, после чего переходим на
страницу соответствующей услуги, на которой отображаются все данные о ней.
Если нужно отредактировать услугу, переходим по ссылке на страницу
редактирования. Для удаления нажимаем на кнопку "удалить"
Добавить услугу можно на странице всех существующих услуг, нажав на кнопку
"добавить".
